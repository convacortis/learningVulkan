# 1. Vulkan triangle

My journey learning Vulkan started with creating a triangle. Vulkan is known to have a steep learning curve and my experience wasn't an exception to this, the setup was arduous but highly rewarding, with more than 1000 lines of code before even getting to anything triangle related. I learned a lot about how the API works, how specific functions are utilised and how the graphics pipeline works.

Throughout the process vulkan-tutorial.com provided an excellent guide explaining every detail
